# AppsRow - Premium Residences

Welcome to **AppsRow**, a modern real estate website showcasing premium residences, landmark properties, and prime locations. Built with HTML, CSS, and JavaScript, this project demonstrates a clean, responsive design with interactive features for property listings and subscriptions.

Live Demo: [https://harshill1912.github.io/appsrow/](https://harshill1912.github.io/appsrow/)

---

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile screens.
- **Hero Section**: Large banner with modern homes and statistics.
- **Property Listings**: Detailed property information including images, location, price, and specifications.
- **Location Map**: Interactive location cards with highlights for prime locations.
- **Testimonials & Partners**: Reviews and partner logos section.
- **Subscription Section**: Collect user email and budget for upcoming listings.
- **Footer & Branding**: Clean footer with company info and links.

---

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (for interactive features)
- Google Fonts (Rubik)
- GitHub Pages (for live deployment)
